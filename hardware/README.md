# Hardware Design

This directory contains all hardware design files for the PET Filament Extrusion System.

## KiCad Version

This project uses KiCad 7.x or later. Please ensure you have a compatible version installed.

## Directory Structure

- `pcb/` - PCB layout files and design
- `schematics/` - Circuit schematics
- `bom/` - Bill of Materials and component lists

## Getting Started

1. Install KiCad from https://www.kicad.org/
2. Open the project files in the respective directories
3. Review the schematics before making any modifications
4. Refer to the BOM for component specifications
